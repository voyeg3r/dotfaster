O plugin tabular faz tabulações de modo bem interessante no vim


    one = 1
    two = 2
    three = 3
    four = 4

    :Tab /=<Enter>

var video = {

    :Tab /:\zs<Enter>

Para tabular o trecho abaixo

width: 215px;
height: 22px;
margin-top: 3px;
background-color: white;
border: 1px solid #999999;


:Tab /:\zs

width:             215px;
height:            22px;
margin-top:        3px;
background-color:  white;
border:            1px solid #999999;


# como alinhar isto

    john,betty,wally,beth
    walter,george,thomas,john
    herbert,bob,petty,mick`

Para ficar assim:

    john,     etty,   ally,   beth
    walter,   eorge,  homas,  john
    herbert,  ob,     etty,   mick`

Basta usar:

  :Tabularize /,zs
